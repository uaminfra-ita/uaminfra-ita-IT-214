# 📁 Arquivos do Plano de Ensino

Coloque os arquivos do plano de ensino nesta pasta:
- `plano-ensino.pdf`
- `cronograma.tex`
- Outros documentos relacionados

Depois, adicione os links em `data/courseData.js`:

```javascript
files: [
  { name: "Plano de Ensino", url: "/files/plano-ensino.pdf" },
  { name: "Cronograma", url: "/files/cronograma.tex" },
],
```
