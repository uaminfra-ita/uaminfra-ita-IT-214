## Recomendações

- **Formato**: JPEG ou PNG
- **Tamanho**: Aproximadamente 300x300 pixels ou superior
- **Resoluição**: Mínimo 72 DPI
- **Compressão**: Use ferramentas como TinyPNG para otimizar

As imagens são carregadas automaticamente em `data/courseData.js`

Exemplo:
```javascript
instructors: [
  {
    // ...
    image: "/images/instructor1.jpg",
  },
],
```
