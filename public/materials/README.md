# 📁 Materiais de Aula

Coloque os materiais de aula nesta pasta:
- `aula1-introducao.pdf`
- `aula2-conceitos.pdf`
- `aula3-regulacoes.pptx`
- Notas de aula
- Slides

Depois, adicione os links em `data/courseData.js`:

```javascript
lectures: [
  { title: "Aula 1 - Introdução", url: "/materials/aula1-introducao.pdf" },
  { title: "Aula 2 - Conceitos", url: "/materials/aula2-conceitos.pdf" },
],
```
